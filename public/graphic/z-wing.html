3D Computer Animation Project

The Z-Wing animation for the game project is a part of the 3D computer animation subject project. This project primarily emphasizes modeling, texturing, rigging, and animation for games.
The challenge lies in creating a new model for the vehicle while ensuring that both the animation and model are not overly detailed to maintain optimal game performance.
